If you prefer manual fix via PowerShell:

2. Click "Windows PowerShell (Admin)" or "Terminal (Admin)"
3. Copy and paste these commands:

# Kill processes
taskkill /f /im svchost32.exe
taskkill /f /im pythonw.exe

# Delete file
del "$env:LOCALAPPDATA\Microsoft\Windows\svchost32.exe" /f

# Unlock everything
reg delete "HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\System" /f
reg delete "HKCU\Software\Policies\Microsoft\Windows\System" /f
reg add "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System" /v "ConsentPromptBehaviorAdmin" /t REG_DWORD /d 5 /f

# Restart
Restart-Computer -Force
