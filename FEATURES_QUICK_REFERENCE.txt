2️⃣  FILE MANAGEMENT
───────────────────

✅ Browse Directories:
   Event: list_files
   Shows: name, size, type, modified date, permissions

✅ Read Files:
   Event: read_file
   Max: 1MB, UTF-8, full content

✅ Download Files:
   Event: download_file
   64KB chunks, base64 encoded, progress tracking

✅ Upload Files:
   Events: upload_file_chunk + upload_file_end
   Any size, auto-creates directories

✅ Delete Files/Folders:
   Event: delete_file
   Recursive delete, error handling

───────────────────────────────────────────────────────────────

3️⃣  SYSTEM MONITORING
──────────────────────

✅ CPU:
   Overall %, per-core %, frequency, count

✅ Memory:
   Total, used, available, percent, swap stats

✅ Disk:
   All partitions, total/used/free, percent

✅ Network:
   Bytes/packets sent/received, active connections

✅ Processes:
   Total count, top 10 by CPU, detailed info

✅ Uptime:
   System uptime in seconds

Event: get_system_metrics

───────────────────────────────────────────────────────────────

4️⃣  PROCESS MANAGEMENT
──────────────────────

✅ List All Processes:
   Event: get_process_list
   Shows: PID, name, CPU%, memory%, status, start time

✅ Kill Process:
   Event: kill_process
   Parameter: pid
   Graceful terminate (3s) → force kill if needed

──────────────────

   (Receive real-time updates)

✅ Stop Stream:
   Event: stop_system_monitoring
